# Roblox-Studio-AI-Helper

To Install

Move the file " RobloxPlugin.lua " into your roblox plugins folder, by going to plugin<plugins folder and then copying that file in there.

To get this working, double click on the exe, paste your gemini api key in and then go to plugins<RobloxAI Builder<Toggle ai connection.
then back to the web page the exe opened and type what you want in the game!
